## Overview

**OCA** is a high-tech, cyberpunk-themed calendar application designed to help you dominate your schedule. It specifically targets Indonesian holidays, providing advanced tools to optimize your leave and maximize your time off.

## Features

*   **Cyberpunk Aesthetics**: A visually stunning, "overengineered" UI with neon accents, glassmorphism, and immersive sound effects.
*   **Interactive Year View**: A comprehensive view of the entire year, highlighting National Holidays and Collective Leave (Cuti Bersama).
*   **Leave Optimizer**: An intelligent algorithm that analyzes your schedule to recommend the best days to take leave, bridging gaps between holidays for maximum vacation time.
    *   **Customizable Strategy**: Set your "Minimum Total Off" to tailor recommendations to your needs.
*   **High-Quality Export**: Generate and download a high-resolution PNG poster of your personalized calendar and leave plan.
*   **Local First**: Runs entirely in your browser.

## Tech Stack

*   **Framework**: React 18
*   **Build Tool**: Vite
*   **Styling**: Tailwind CSS
*   **Export**: html2canvas

## Getting Started

### Prerequisites

*   Node.js (v16 or higher)

### Installation

1.  Clone the repository (if applicable) or navigate to the project directory.

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:3000`.
